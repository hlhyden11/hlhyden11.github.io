U.S. Power Sources

This map allows the user to click anywhere on the map to find out what power plants are located within 500 miles. Then, click on the plant to find out the name, fuel source and distance from the location where you clicked

Javascript functions were used to create "Click" events and a math function was used to create a radius around the location of the plant.

  • See the map:Hydro,Solar and Nuclear Power
  • Data Source:Open Source Location with Power Plant Data from Around World. This Link Takes You Directly to the U.S. Page

Lexington Buildings By Age

This map shows the age of buildings in Downtown and on the Northside of Lexington. Darker red hues represent older buildings and darker blues reflect the most recent construction. Additionally, the pale color represents missing PVA building age data.

The map was created in ArcGIS and CartoDB.

  • See the map:Lexington Buildings by Age
  • Data Source:Open Data Lexington

Lexington Buildings By Age
